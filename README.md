Aufgabe 1:	Weihnachtskekse
Rund ausgestochene Weihnachskekse werden auf der Oberseite mit Schokolade übergossen.
Schreibe ein Programm, welches die Anzahl number_biscuits und den Durchmesser der Kekse (in cm) diameter_biscuit einliest und die Gesamtfläche für die Schokoladeglasur berechnet.
•	„Für die Glasur aller xxx Kekse wird Schokolade für yyy cm² benötigt.“


Aufgabe 2:	Christbaumschmuck
Ein Christbaum wird mit Kerzen und Kugeln geschmückt. Eine Kerze kostet € 1,50 und eine Kugel kostet € 4,90.
Schreibe ein Programm, welches eine Anzahl von Kerzen candles und eine Anzahl von Kugeln balls einliest und folgendes berechnet bzw. ausgibt:
•	„Die Gesamtkosten für den Christbaumschmuck betragen xxx Euro.“
•	„Auf dem Baum befinden sich weniger als 10/10 oder mehr Kerzen.“
•	„Auf dem Baum befindet sich eine gerad/ungerade Anzahl Kugeln.“


Hinweise:
•	Stelle jeder Eingabe eine Eingabeaufforderung voran.
•	Lege für jede Variable einen sinnvollen Datentyp fest.
•	Leite jede Aufgabe mit einer eigenen Ausgabezeile „Aufgabe 1: Weihnachtskekse“ bzw. „Aufgabe 2: Christbaumschmuck“ ein.
